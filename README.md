## Projeto API REST Library

## Sobre o Projeto
Este projeto foi desenvolvido com o objetivo de praticar os conceitos aprendidos no curso "API REST e Spring Boot: Aprenda do Zero e na Prática" da Udemy. A API simula o funcionamento de uma biblioteca, fornecendo diversos endpoints para operações como:
- GET: Obtenção de informações sobre livros e autores.
- POST: Cadastro de novos livros e autores.

## Tecnologias e Conceitos Aplicados
- Spring Boot
- **Swagger** para documentação da API
- **Spring Security** e autenticação com JWT
- **Spring Data JPA** para persistência de dados
- Testes Unitários
- Uso de DTOs para transferência de dados
